      The basics
      
How tokenization works

      
LLMs don't see words — they see tokens. Understanding this is the key to controlling your API bill.

    


    

      

        
01

        
Text is split into tokens

        
Each model uses its own tokenizer to break text into subword units. The same words can produce different token counts on different models.

        

          
Example: "You are a helpful assistant"

          

            You
            are
            a
            help
            ful
            assist
            ant
          

          
= 7 tokens (Claude) · 7 tokens (GPT-4) · 6 tokens (Llama)

        

      


      

        
02

        
Tokens × price = API cost

        
Models charge per 1 million input and output tokens separately. Input is usually cheaper than output — prompts and system messages are input tokens.

        

          

            

              1,000 tokens
              ×
              $3.00 / 1M
              =
              $0.003
            

            
Claude 3.5 Sonnet input pricing

          

        

      


      

        
03

        
Optimize to save money

        
Shorter prompts → fewer tokens → lower cost. Tokenia shows you exactly how many tokens you can reclaim by cleaning redundant whitespace and boilerplate.

        

          

            

              Before
              2,480 tokens · $0.0074
            

            
↓ clean

            

              After
              1,620 tokens · $0.0049
            

            
Saved 34% · $0.0025 per call
