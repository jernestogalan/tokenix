Privacy & Security

Built with your data security as priority #1

Most token calculators log your prompts. We don't — and we've verified it in our code.

HTTPS — Text Never Stored

Your text is sent to our server over HTTPS for accurate token counting, then immediately discarded. We don't store, log, or retain your prompts.

Zero Data Retention

Your text is processed in memory and discarded after each request. No database writes, no file storage of your content. Verified in our source code.

No Third-Party Trackers

No Google Analytics, no ad pixels, no third-party cookies. We use lightweight first-party analytics that log only event names and page URLs — never your text.

Hardened Security Headers

All connections via HTTPS/TLS. Security headers enforced: HSTS, CSP, X-Frame-Options, nosniff. Read our security page →
