Frequently asked questions

What counts as a token?

A token is a chunk of text as processed by a language model. For English text, one token is roughly 3–4 characters or 0.75 words. For OpenAI models, we use the official tiktoken tokenizer for exact counts. For other providers, we use a calibrated heuristic and clearly label results as "~Estimated".

How accurate are the cost estimates?

Cost estimates are based on publicly listed API pricing. We update prices regularly, but providers can change rates without notice. Always verify against your provider's pricing page before making financial decisions. Token counts for non-OpenAI models can vary ±5–15% from actual counts.

Can I use my own API key?

Not yet — this is planned as a Pro/Team feature. Using your own key would allow us to run the provider's official tokenizer for exact counts on all models.

Is my text stored or used for training?

No. All text and files are processed in-memory and discarded immediately after analysis. We never store, log, or use your content.

What file formats are supported?

PDF, DOCX, TXT, Markdown, and most programming language files (.py, .js, .ts, .go, .rs, .java, .kt, .swift, .c, .cpp, .cs, .php, etc.). HTML and JSON are also supported. Max file size: 2 MB on Free, 20 MB on Pro, 50 MB on Team.
