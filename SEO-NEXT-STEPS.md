# SEO Next Steps for Tokenia

Follow these steps in order. Each takes 5–30 minutes.

## Immediate (Do This Week)

### 1. Submit to Google Search Console ⏱ 10 min
1. Go to search.google.com/search-console
2. Add property → URL prefix → `https://tokenia.live`
3. Verify via HTML file: download file → place in `public/` → deploy → verify
4. Submit sitemap: Add sitemap → `https://tokenia.live/sitemap.xml`
5. Submit blog sitemap: `https://tokenia.live/blog/sitemap.xml`

### 2. Submit to Bing Webmaster Tools ⏱ 5 min
1. Go to bing.com/webmasters
2. Import from Google Search Console (instant)
